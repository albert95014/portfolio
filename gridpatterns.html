Grid Patterns

An exercise in pattern generation and pixel manipulation using p5, inspired by a few Reddit posts I came across.

2019

Breakdown

Each cell takes a number of simple patterns (all 25 shown below), and overlays them to create something more complex. I played with the color and size, as well as the number of patterns each cell has, to simulate the pixels in some photos I took. This created a pointilist-esque apperance to the images, with each "pixel" being a miniature complex pattern.
